# Model Release Checklist

## Weights and artifacts
- Save final adapter weights.
- Save tokenizer files with the adapter.
- Save training config and run metadata.
- Save evaluation report JSON.

## Public repo
- Include setup instructions.
- Include dataset schema and prep commands.
- Include training command and config examples.
- Include evaluation command and metrics definition.
- Include demo launch command.

## Kaggle submission assets
- Kaggle writeup link
- YouTube demo link
- Public code repository link
- Live demo link
- Cover image and media gallery assets

## Evidence package
- One held-out benchmark table
- One qualitative base-vs-tuned example
- One screenshot of the live demo
- Clear note that the teacher approves the output before classroom use
